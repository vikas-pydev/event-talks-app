### <p align="center" style="color: #a777e3;">🏃‍♀️ How to Run</p>

<div align="center">

1.  **Navigate to the directory:**
    ```bash
    cd tech-event-website
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the server:**
    ```bash
    npm start
    ```

4.  **Open in your browser:** `http://localhost:3000`
